Folder¶

Description¶

A folder for (purely) organizational purposes. Variables defined in a folder will be displayed as an expandable group in MWClient’s variables window.

Signature¶

folder

Alias

group (MWEL only)

Placement¶

Allowed at top level:

Yes

Allowed children:

Calibrator, Filter, IODevice, List Replicator, Optimizer, Range Replicator, Resource, Selection Variable, Sound Stimulus, Stimulus, Stimulus Group, Variable
