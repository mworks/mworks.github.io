Unload Stimulus¶

Description¶

Explicitly unload a stimulus object, thereby releasing any memory or other resources used by the stimulus. This is useful when the deferred parameter on the stimulus is set to yes or explicit.

Signature¶

action/unload_stimulus

Required Parameters¶

stimulus¶

Stimulus to unload

Placement¶

Allowed at top level:

No

Allowed parent:

Block, Conditionally Execute Actions (if), List, Protocol, Repeat Actions (while), Schedule Actions, Task System State, Trial, Unconditionally Execute Actions (else), Variable
