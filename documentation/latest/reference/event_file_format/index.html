Event File Format¶

The following sections describe the on-disk format of MWorks event files. This information is provided both for reference and to assist users who wish to develop bespoke software for reading event files.

Most users should use MWorks’ standard tools for reading event files. For details on how to do so, see Analyzing Experimental Data.
