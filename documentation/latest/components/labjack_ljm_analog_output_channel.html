LabJack LJM Analog Output Channel¶

Available on macOS

Description¶

Analog output channel on a LabJack LJM Device

Signature¶

iochannel/labjack_ljm_analog_output

Required Parameters¶

line¶

Examples:
DAC0
dac1

Analog output line name

value¶

Variable from which to read output voltage values. Assigning a value to the variable causes the output voltage to be changed immediately.

Placement¶

Allowed at top level:

No

Allowed parent:

LabJack LJM Device
