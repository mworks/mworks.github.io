TRACKPixx Load Pupil Size Calibration¶

Available on macOS

Description¶

Load the pupil size calibration saved to a TRACKPixx Device

Signature¶

action/trackpixx_load_pupil_size_calibration

Required Parameters¶

device¶

Name of the TRACKPixx device

Placement¶

Allowed at top level:

No

Allowed parent:

Block, Conditionally Execute Actions (if), List, On Exit, Protocol, Render Actions, Repeat Actions (while), Schedule Actions, Task System State, Trial, Unconditionally Execute Actions (else), Variable
