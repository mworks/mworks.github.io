Stop Sound¶

Description¶

Stop a sound.

If the sound is playing or paused, playback will end immediately, and the next playback will start at the beginning of the sound. If the sound is neither playing nor paused, this action does nothing.

Signature¶

action/stop_sound

Required Parameters¶

sound¶

Name of the sound

Placement¶

Allowed at top level:

No

Allowed parent:

Block, Conditionally Execute Actions (if), List, On Exit, Protocol, Render Actions, Repeat Actions (while), Schedule Actions, Task System State, Trial, Unconditionally Execute Actions (else), Variable
