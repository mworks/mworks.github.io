DATAPixx Analog Output Channel¶

Available on macOS

Description¶

Analog output channel on a DATAPixx Device

Signature¶

iochannel/datapixx_analog_output

Required Parameters¶

channel_number¶

Examples
0
3

Analog output channel number

value¶

Variable from which to read output voltage values. Assigning a value to the variable causes the output voltage to be changed immediately.

Optional Parameters¶

device_time_nanos¶

Variable in which to store the device time (in nanoseconds) when each value is output by the device

Placement¶

Allowed at top level:

No

Allowed parent:

DATAPixx Device
