Sound Stimulus¶

Description¶

A simple WAV file based sound stimulus

Signature¶

sound/wav_file

Required Parameters¶

path¶

Examples
sounds/sound1.wav
/path/to/sound.wav

Filesystem path to the sound file

Optional Parameters¶

amplitude¶

Default

1.0

Controls the volume at which the sound plays. 1.0 is full volume, while 0.0 is silent.

Placement¶

Allowed at top level:

Yes

Allowed parent:

Folder, List Replicator, Range Replicator
