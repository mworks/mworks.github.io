Timer Expired Transition¶

Description¶

A transition that goes to the target state if a specified timer has expired

Signature¶

transition/timer_expired

Required Parameters¶

timer¶

Name of the timer

target¶

Name of the target state

Placement¶

Allowed at top level:

No

Allowed parent:

Task System State
