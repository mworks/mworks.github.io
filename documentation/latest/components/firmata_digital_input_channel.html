Firmata Digital Input Channel¶

Description¶

Digital input channel on a Firmata Device

Signature¶

iochannel/firmata_digital_input

Required Parameters¶

pin_number¶

Example:

13

Input pin number

value¶

Variable in which to store input state (true or false)

Placement¶

Allowed at top level:

No

Allowed parent:
