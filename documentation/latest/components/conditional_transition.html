Conditional Transition¶

Description¶

A transition that goes to the target state if a specified condition expression is true

Signature¶

transition/conditional

Required Parameters¶

condition¶

Condition expression

target¶

Name of the target state

Placement¶

Allowed at top level:

No

Allowed parent:

Task System State
