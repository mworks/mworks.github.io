Dequeue Stimulus¶

Description¶

Remove a stimulus from the display queue, so that it will no longer be displayed after the next Update Stimulus Display action is issued

Signature¶

action/dequeue_stimulus

Alias:

action/hide_stimulus

Required Parameters¶

stimulus¶

Name of the stimulus

Placement¶

Allowed at top level:

No

Allowed parent:

Block, Conditionally Execute Actions (if), List, On Exit, Protocol, Render Actions, Repeat Actions (while), Schedule Actions, Task System State, Trial, Unconditionally Execute Actions (else), Variable
